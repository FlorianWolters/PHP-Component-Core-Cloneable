Class CloneNotSupportedException

Thrown to indicate that the __clone method in a class has been called to clone an object, but that the object's class does not implement the FlorianWolters\Component\Core\CloneableInterface.

Applications that override the __clone method can also throw this exception to indicate that an object could not or should not be cloned.
